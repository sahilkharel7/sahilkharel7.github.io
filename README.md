# Sahil Kharel — Personal Portfolio

Personal portfolio website built with React, TypeScript, GSAP, Three.js, and WebGL.

## Tech Stack

React, TypeScript, Vite, GSAP, Three.js, WebGL, CSS

## Getting Started

```bash
npm install
npm run dev
```

## Notes

- GSAP Club plugins (ScrollSmoother) are required for production hosting. The trial version is included for development only. See [GSAP Installation](https://gsap.com/docs/v3/Installation/) for details.
- Project images in `/public/images/` need to be added for the Work and TechStack sections.
